# MoonPair — Marketing Website

Marketing site for MoonPair, a couples app for sending love, saving memories, and feeling close.

Built with React 19 + Vite.

## Structure

- `src/components/` — one folder per section (Navbar, Hero, EverythingYouCanDo, FeatureCards, CTA, Footer), each with its own `.jsx` and `.css`.
- `src/assets/` — live images used by the site.
- `src/assets/_archive/` — raw source images and earlier design revisions, kept for reference/revert.
